## 使用方法
1. env.example.php改为env.php
然后修改里面常量值的为自己的信息
2. php kx cron:bot coin/btc
btc可以更换其他币种，当前仅支持火币usdt区



## 今后的方法
请保留账户中有足够的usdt
每笔交易为50usdt 可以开多窗口运行
2. php kx migrate:up 导入数据库
3. php kx robot:start 运行
4. php kx robot:stats 查看统计
